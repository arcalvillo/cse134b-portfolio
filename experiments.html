Why MathML wasn't usable until recently

Although MathML has been part of the HTML specification for over two decades, developers largely couldn't rely on it until recently. According to caniuse.com, browser support was historically inconsistent and incomplete: Firefox supported MathML natively for years, but Chrome, Edge, and Safari either lacked native support entirely or offered only partial implementations, forcing developers to rely on JavaScript libraries like MathJax to render equations reliably. It wasn't until Chrome 109 (released in early 2023) added native MathML Core support that all major browsers finally converged on a shared, native rendering path — making it practical to use MathML directly in production for the first time.
