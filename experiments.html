Why was MathML wasn't usable until recently?

Even though MathML has technically been part of the HTML spec for a really long time, developers basically couldn't use it until recently because browser support was all over the place. When I checked caniuse.com, I saw that Firefox supported MathML for years, but Chrome, Edge, and Safari either didn't support it at all or only supported parts of it. This meant that if you tried to write an equation in MathML, it might look great in Firefox but completely break in Chrome, so most developers just avoided it and used JavaScript libraries like MathJax instead to render math. Chrome didn't add real native support until version 109 in early 2023, which is when it finally became something you could actually use across all major browsers without a fallback.
